Spring Boot REST API sample, that parses a xml file based on a liquid template structure.

## Getting Started

### Prerequisites

Use maven package manager to run the application.
| Prerequisites | Version |
|---|---|
| OpenJDK | 19 |
| Maven| ~ 3.9.0|

### Run 
```bash
./mvnw spring-boot:run
```
